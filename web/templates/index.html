The Suaves

A group of passionate data analysts. The Suaves are individuals coming from different technological backgrounds who all have one thing in common; the desire to acquire more knowledge. This project is a display of skills obtained through data coding and visualization.

ETL Project

The aim of this project is to compare mobile apps on both Apple and Android devices. We accomplished this by pulling raw data from Kaggle; cleaning and transforming it. We then created a flask API that serves up JSON representation of mobile app data. After that a flask API was created that takes in a route variable and returns various JSON depending on at least one query parameter.
